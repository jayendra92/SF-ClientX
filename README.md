# SF-ClientX
SF-ClientX is a Windows-based IT Asset Inventory and Endpoint Monitoring solution


How SF-ClientX Works
Step 1: Install the SF-ClientX Server.

Step 2: Install the Client Agent on each Windows computer.

Step 3: The Client Agent automatically collects hardware and software information.

Step 4: The collected information is securely sent to the SF-ClientX Server.

Step 5: The Server stores the information in PostgreSQL.

Step 6: The web dashboard displays all inventory information.

Step 7: Whenever hardware or software changes are detected, the Client Agent automatically updates the server, ensuring the inventory remains accurate.
